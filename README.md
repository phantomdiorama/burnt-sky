# Hurry Asteroid

> "Don't they know it's the end of the world?"

## Install

run `install.sh`

## Use

Add user feed with:

```
hurryasteroid https://bsky.app/profile/exampleuser.bsky.social
```

Build page with:

```
hurryasteroid
```

Schedule with cron:

```
0 * * * *  ~/.local/bin/hurryasteroid
```
